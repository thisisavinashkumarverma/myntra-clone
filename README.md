# Myntra Clone

A front-end **Myntra** clone that replicates the UI of the popular Indian fashion e‑commerce platform, focusing on layout, navigation, and basic shopping interface elements.[page:1]

## Live Demo

- GitHub Pages: https://thisisavinashkumarverma.github.io/myntra-clone/

## Features

- Top navigation bar with categories: Men, Women, Kids, Home, Beauty, Genz, Studio, etc.[page:1]  
- Search input field for products, brands, and more (UI only).[page:1]  
- Header icons/links for Profile, Wishlist, and Bag (non-functional placeholders).[page:1]  
- Footer sections including:
  - Online Shopping links (Men, Women, Kids, Home, Beauty, Genz, Gift Cards, Myntra Insider).[page:1]
  - Useful Links (Blogs, Careers, Site Map, Corporate Information, Whitehat, Cleartrip, Myntra Global).[page:1]
  - Customer Policies (Contact Us, FAQ, T&C, Terms of Use, Track Orders, Shipping, Cancellation, Returns, Privacy Policy, Grievance Redressal).[page:1]
- Popular searches section with many fashion-related quick links (Makeup, Dresses for Girls, T‑Shirts, Sandals, Handbags, Watches, Shoes, etc.).[page:1]
- Footer note for 100% original guarantee and Myntra/Flipkart branding text.[page:1]

## Tech Stack

## Getting Started

1. Clone the repository:
git clone https://github.com/thisisavinashkumarverma/myntra-clone.git
2. Navigate into the project folder:
cd myntra-clone
3. Open `index.html` directly in your browser or use a local server (e.g., VS Code Live Server).
