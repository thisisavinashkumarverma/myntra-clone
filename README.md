# Myntra Clone

A responsive web clone of Myntra, built for learning and demonstration purposes.

## Features

- Home page with dynamic banners and featured products
- Product listing with filtering and sorting
- Product detail pages with information display
- Shopping cart experience
- Responsive design for desktop and mobile
- Built using HTML, CSS, and JavaScript

## Getting Started
